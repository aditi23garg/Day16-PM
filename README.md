# 📊 Exploratory Data Analysis (EDA) Assignment

## 📌 Overview

This project focuses on **Exploratory Data Analysis (EDA)**, which is the process of analyzing and understanding data before applying machine learning models.
It includes data cleaning, visualization, and extracting meaningful insights using Python.

This assignment is part of **AI/ML coursework** and helps in building a strong foundation in data analysis.
